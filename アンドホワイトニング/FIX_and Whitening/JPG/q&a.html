ホワイトニングの効果について

  • Q,どうして歯が黄色くなるのですか？

    +

    A,コーヒー、紅茶、カレーやワインなどの濃い色の飲食物による色素沈着が原因です。 また加齢によって歯の表面の半透明の部分（エナメル質）が薄くなり、歯の内側の黄色い部分（象牙質）が肥厚し透けて見えることで黄色く見えます。

  • Q,歯医者で行うホワイトニングと美容サロンで行うホワイトニングの違いは何ですか？

    +

    A,オフィスホワイトニングと比較し、安価・安全で且つ痛みもありません。

  • Q,1回の施術にどのくらい時間がかかりますか？

    +

    A,初回はカウンセリングも含めて約30分程度お時間をいただいております。 2回目以降は20分程度が目安となっております。

  • Q,どのくらい白くなりますか？

    +

    A,セルフホワイトニングでは歯の表面の着色汚れを分解し歯を本来の白さに戻します。 差し歯や入れ歯、詰め物、歯の神経を抜いたことにより歯が変色している場合や遺伝によりもともと色素沈着がある場合などは効果が期待できないこともあります。

  • Q,歯が白くなるまでどのくらいかかりますか？

    +

    A,人それぞれ白さの基準が違うため個人差はありますが1回の施術でも効果はあります。 1回で効果を感じる方も多くいますが、1回目で効果を感じられなかった方でも回数を重ねるとさらに白くなる方が多いです。 その方のもともとの歯の色のトーンや施術回数などによって変わってきます。 また、喫煙やコーヒーなど色の濃い飲食物を日常的に摂取される方はもう少し回数が必要な場合もあります。

  • Q,施術すればずっと白いままでいられますか？白さはどのくらいもちますか？

    +

    A,日常的に飲食物や喫煙などによって歯の着色は起こってしまいます。 効果を持続するために定期的に施術を受けることをおすすめします。 最初は集中的に継続して施術し効果を感じられたら、1～3ヶ月ごとにメンテナンスすることで白さを保つことができます。

ホワイトニングができる条件

  • Q,歯の本数を限定してホワイトニングできますか？

    +

    A,口を開けた際に見える範囲全体をホワイトニングするため歯の本数を限定してホワイトニングすることは難しいです。

  • Q,歯の裏側や奥歯もホワイトニングできますか？

    +


    A,難しいです。
    基本的にホワイトニングの効果はホワイトニング溶液を塗布して光が当たる範囲なので歯の裏側はできません。
    奥歯はホワイトニング溶液を塗布しにくく光も当たりにくいため前歯ほど効果が出ない可能性があります。

  • Q,矯正中でもホワイトニングできますか？

    +


    A,はい、可能です。
    通常のブランケット矯正（歯の表面に装置をつける方法）をされている場合は矯正装置がついている部分にホワイトニング溶液が塗布できないため効果はあまり期待できません。
    見えている部分だけでよろしければ不可能ではありません。
    その他の歯の表面が見える方法で矯正されている場合は問題なく施術していただけます。

  • Q,差し歯でもホワイトニングできますか？

    +


    A,はい、可能です。
    差し歯の表面についた着色汚れを落とすことはできますが、経年により全体的に変色してしまった差し歯を白くする効果は期待できません。
    天然歯に比べ人工歯は変色、着色しやすいため日頃からのケアをおすすめいたします。

  • Q,銀歯があってもホワイトニングできますか？

    +


    A,はい、可能です。
    ホワイトニングに支障はありません。

  • Q,妊娠中でもホワイトニングできますか？

    +


    A,お医者様への確認をお願いいたします。
    お身体に害のあるものは使用をしていませんが、ケアを行う間は、一定の姿勢を保たなければならないなどの不便や、妊婦の方は通常よりも体内に入れるものに対して敏感にな
    っている可能性が高いことから、ご利用をお断りする場合がございます。

and Whiteningのご利用について

  • Q,予約はどこからしたらいいですか？

    +

    A,各お取り扱い店舗にお電話などでご予約が可能です。

  • Q,予約の変更やキャンセルはできますか？

    +

    ご予約のキャンセル、変更はご予約された各店舗へ連絡をお願いします。

  • Q,持ち物はいりますか？

    +


    ホワイトニングの施術に使用するものは全てご用意していますのでホワイトニングに関しての持ち物は必要ありません。
    お支払いに必要なもののみご持参ください。

  • Q,保険は適用されますか？

    +

    保険は適用されません。

その他のご質問

  • Q,なぜ美容院で限定しているのですか？

    +


    A,業界初のビジネスモデルです。
    美容院は全国に25万店舗もあるため全国どこでもホワイトニングができます。
    美容院の空きスペースを利用することで待ち時間などで気軽にホワイトニングが可能です。

  • Q,なぜ歯医者で高いものが安くできるんですか？

    +


    歯医者で行うオフィスホワイトニングでは過酸化水素、過酸化炭素という高濃度の溶液を使用しLEDライトを照射します。
    専門の歯科医でないと扱うことができない薬品であり、歯科医が施術をします。そのため高額になります。
    一方で、美容サロンなどで行うセルフホワイトニングでは酸化チタンという溶液を使用しLEDライトを照射します。
    誰でも手に入れることができ、安心安全な薬剤です。
    またセルフでホワイトニングすることで人件費もかからないため価格を抑えることが可能です。
